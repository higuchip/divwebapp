Div Web (Teste)

Instrução de uso:

DIV WEB: aplicativo web desenvolvido em Javascript para determinação do Índice de Diversidade de Shannon (H'), da Equabilidade de Pielou (J), do Índice de Simpson (D), do Índice de Diversidade de Simpson (1-D) e do Índice de Simpson Inverso (1/D).

O arquivo de entrada deve ser no formato csv, com duas colunas. Uma coluna deve conter as espécies e a outras, necessariamente nomeada como n, deve conter o número de indivíduos de cada espécie.

Arquivo exemplo disponível aqui.
